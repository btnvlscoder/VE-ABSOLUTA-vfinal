\# VE ABSOLUTA - Backend (Módulo de Orquestación e IA)



Proyecto de título para la carrera de Analista Programador. 

Sistema de detección de imágenes generadas por Inteligencia Artificial.



\## Tecnologías

\* \*\*Lenguaje:\*\* Kotlin + Java 25

\* \*\*Framework:\*\* Spring Boot 4.0.5

\* \*\*Base de Datos:\*\* PostgreSQL 17

\* \*\*Motor de IA:\*\* Python 3.x (OpenCV / TensorFlow)



\## Instalación

1\. Clonar el repositorio.

2\. Configurar `application.properties` con tus credenciales de PostgreSQL.

3\. Instalar dependencias de Python:

&#x20;  ```bash

&#x20;  pip install -r ai\_engine/requirements.txt
